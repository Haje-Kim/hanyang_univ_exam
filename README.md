## 기능 (v1.1)

- 🧠 **5단 개념 빌드업** — 나열이 아니라 `Hook(왜 배우나) → 직관 → 비유 → 정의 → 메커니즘`으로 이해시키는 설명
- 🎬 **3Blue1Brown식 모핑 애니메이션** — 알고리즘 전개를 캔버스로 부드럽게 시각화(정렬·경사하강·min-conflicts·GNN message passing·ROC·합성곱). 자체 경량 엔진(외부 라이브러리 0)으로 **완전 오프라인**
- 🔤 **영어(한글) 병기** — 전문 용어를 `English(한글)`로 병기해 원문 검색·이해를 동시에
- 📝 **실전 예상문제 + 모범답안** — 과목별 출제 양식(서술형 full-text / 증명 / 객관식 오답해설 / 매칭표) + 흔한 실수
- 📚 **강의 핵심 노트** — [`lecture-notes/`](lecture-notes/) 에 과목별 핵심 추출 마크다운
- ✅ **진도 저장**(localStorage) · 🌓 다크/라이트 · 📱 반응형 · ♿ `prefers-reduced-motion` 대응
