# 한양대 AI융합대학원 — 기말 시험대비 학습 허브

강의자료(PDF)와 강의 노트를 분석해 만든 **인터랙티브 시험대비 학습 자료** 모음입니다.
각 과목을 **top-down 로드맵 → 핵심 개념 drill-down** 구조로 정리하고, 실세계 시나리오·단계별 풀이 예제·시험 포인트를 담았습니다.

## 시작하기

`index.html`을 브라우저로 열면 4과목 학습 허브(랜딩 페이지)가 나옵니다. 각 과목 카드를 클릭해 학습을 시작하세요.
모든 페이지는 **단일 HTML · 오프라인 동작**(외부 의존은 수식 렌더용 KaTeX CDN뿐, 실패 시 원본 수식 폴백)입니다.

## 기능

- 🌳 **MECE 트리맵** — 첫 화면에서 과목 전체를 빠짐없이·겹침없이 Top-Down으로 (잎 클릭 → 드릴다운)
- 🗺️ **로드맵 + 추천 학습경로** — 구조(공간)와 의존도 위상정렬 동선(시간)
- 🌍 **실세계 시나리오** — 추상 개념을 실제 사용 사례로 연결
- ✏️ **단계별 풀이 예제** — 학술 정의만이 아니라 구체 입력 → 단계별 풀이 → 답 (스테퍼)
- 📰 **논문 포스터** — 핵심 논문을 terminology·concept·breakthrough로 (의생명: DNABERT·Deep Learning·GAT)
- 📝 **예상 문제 + 모범 답변** — 시험 출제 가이드 기반, 답 가림 → 펼침 셀프테스트
- ✅ **진도 저장** — 브라우저 localStorage (개념별 완료 체크)
- 📱 **반응형** — 아이패드·모바일 대응 · 🌓 다크/라이트 · 🔎 개념 검색

## 비고

본 자료는 **개인 학습용 정리·해설**이며, 강의 콘텐츠의 저작권은 각 담당 교수에게 있습니다. 원본 강의 PDF·논문은 저장소에 포함하지 않았습니다(학습 요약·해설만 공개).
