# 인터랙티브 논문 포스터 — 의생명 AI 심화

의생명과학 AI 과목의 핵심 논문 3편(DNABERT · Deep Learning(노미나 교수 강의 핵심 논문) · GAT)을 **모듈별 인터랙션을 넣어 분량 3배로 심화**한다. 기존의 단순 "terminology·concept·breakthrough" 3블록 압축을 풀어 각 모듈을 조작 가능한 학습 단위로 만든다.

## 목차
1. 확장 원칙 (왜 3배인가)
2. 포스터 모듈 구조 (7모듈)
3. 논문별 인터랙션 매핑
4. 체크리스트

## 1. 확장 원칙

- **압축 해제 ≠ 물타기.** 분량 3배는 빈 말 추가가 아니라, 각 모듈에 (a) 직관 (b) 정확한 정의·수식 (c) 조작 가능한 인터랙션 (d) 시험 포인트를 더해 채운다.
- **서술형 대비.** 이 과목은 "입력→출력" 서술형, 논문 3편 중심, GNN 식 확정, 주차별 2문제. 포스터는 시험에서 그대로 쓸 수 있는 깊이여야 한다.
- **인터랙션은 개념당 1개 이상.** 읽기만 하는 포스터가 아니라, 핵심 메커니즘마다 simulations.md의 패턴을 끼운다.

## 2. 포스터 모듈 구조 (7모듈)

각 논문 포스터를 아래 7모듈로 전개(기존 3블록 → 7모듈 = 분량 3배의 근거):

| 모듈 | 내용 | 인터랙션 |
|---|---|---|
| ① 문제 정의 | 무엇을 풀려는 논문인가 (입력·출력·태스크) | 입력→출력 예시 토글 |
| ② 선행 용어 | 이해에 필요한 생물/딥러닝 용어 | 용어 호버 정의(content-architect와 연계) |
| ③ 핵심 아이디어 | breakthrough 한 문장 + 직관 | before/after 비교 |
| ④ 아키텍처 | 모델 구조 도식 | 단계별 데이터 흐름 스테퍼 |
| ⑤ 수식·메커니즘 | 핵심 수식 유도 | 메시지패싱/attention 시뮬레이션(simulations §4) |
| ⑥ 결과·의의 | 성능·기여 | 지표 슬라이더 또는 결과 표 하이라이트 |
| ⑦ 시험 포인트 | 예상 서술형 + 모범답변 | 답 가림→펼침 셀프테스트 |

## 3. 논문별 인터랙션 매핑

- **DNABERT** — ④에서 DNA 서열 → k-mer 토큰화 → 트랜스포머 단계 스테퍼. ⑤ self-attention을 토큰 간 화살표 굵기로. ①에서 서열 입력→다운스트림 태스크(프로모터 예측 등) 출력 토글.
- **Deep Learning(기초 논문)** — ④ 레이어별 표현 학습 스테퍼(입력→은닉→출력 활성). ⑤ 역전파 그래디언트 흐름 애니메이션. 과목 기초이므로 용어 모듈(②)을 두텁게.
- **GAT (Graph Attention Network)** — ⑤가 핵심: GNN 메시지 패싱 + attention 가중치 시뮬레이션(simulations §4). 식 $h_v^{(k)}=\sigma(\sum_{u\in N(v)}\alpha_{vu} W h_u^{(k-1)})$ 확정 표기. ④ 노드 임베딩 갱신 스테퍼.

## 4. 체크리스트

- [ ] 논문 3편 모두 7모듈로 전개(기존 대비 약 3배 분량)
- [ ] 각 논문에 인터랙션 ≥ 2개(아키텍처 스테퍼 + 메커니즘 시뮬레이션)
- [ ] GNN/attention 식이 강의 확정 표기와 일치
- [ ] ⑦ 셀프테스트(답 가림→펼침) 동작
- [ ] 오프라인·다크모드·모바일 OK (공통 규칙 준수)
